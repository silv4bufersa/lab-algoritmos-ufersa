# lab-algoritmos-ufersa

Repositório com códigos referentes as disciplinas de Algoritmos e Laboratório de Algoritmos

## Pastas

`aulas/`: Arquivos das aulas de programação ministradas em sala. (a partir da segunda unidade).

`exemplos/`: Todos os exemplos que vemos em sala e que por ventura venham a ser adicionados para consulta ou estudos.

`listas/`: Listas disponibilizadas no SIGAA e suas resoluções.

* Resoluções das listas publicadas após o prazo de entrega ou resolvidas em sala.

`manual-c/`: Manual de consulta primária ao material, contendo exemplos de códigos e definições.
