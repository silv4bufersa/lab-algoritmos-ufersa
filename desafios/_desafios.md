# Desafios de Lógica de Programação: Estruturas de Controle

Esta lista contém 15 exercícios para praticar o uso de `if`, `else`, `while`, `do-while` e `for`.

### 1. Contagem Regressiva com Alerta

Escreva um programa que use um laço `for` para imprimir uma contagem regressiva de 10 até 1. Ao final, se o número for 1, imprima "Fogo!".

### 2. Somatório de Números Positivos

Crie um programa que peça números ao usuário usando `while`. O laço deve parar quando o usuário digitar um número negativo. Ao final, exiba a soma de todos os números positivos digitados.

### 3. Filtro de Números Pares

Dado um intervalo de 1 a 20, use um laço `for` e uma estrutura `if` para imprimir apenas os números que são pares.

### 4. Tabuada Personalizada

Solicite um número ao usuário e, utilizando um laço `for`, exiba a tabuada desse número (de 1 a 10).

### 5. Validação com Do-While

Crie um programa que peça ao usuário para digitar uma nota entre 0 e 10. Utilize `do-while` para repetir a leitura caso o valor digitado seja inválido (menor que 0 ou maior que 10).

### 6. Localizador de Múltiplos

Use um laço `for` para percorrer os números de 1 a 50. Para cada número, verifique com um `if` se ele é múltiplo de 5 e imprima-o na tela.

### 7. Verificador de Senha

Crie um sistema que peça uma senha ao usuário. Use um laço `while` para continuar pedindo a senha até que ele digite o valor correto "1234". Se errar, exiba "Senha Incorreta".

### 8. Média de N Notas

Peça ao usuário para dizer quantos alunos existem na sala. Use um laço `for` para ler a nota de cada aluno e, ao final, calcule e exiba a média aritmética da turma.

### 9. O Dobro até o Limite

Peça um número inicial ao usuário (ex: 2). Use um laço `while` para multiplicar esse número por 2 e exibir o resultado, repetindo o processo enquanto o valor for menor que 1000.

### 10. Quadrado de Asteriscos

Peça um número inteiro N ao usuário. Utilize dois laços `for` aninhados para imprimir um quadrado de asteriscos de tamanho N x N.

### 11. Comparador de Idades na Fila

Em um laço `for` que simula 5 pessoas em uma fila, peça a idade de cada uma. Se a idade for maior ou igual a 60, imprima "Fila Preferencial", caso contrário, "Fila Comum".

### 12. Soma de Ímpares no Intervalo

Peça dois números inteiros ao usuário que representem um intervalo (início e fim). Use um laço `for` e um `if` para somar apenas os números ímpares dentro desse intervalo.

### 13. Menu de Opções Simples

Utilize um laço `do-while` para exibir um menu: "1. Olá", "2. Bem-vindo", "0. Sair". O programa deve continuar executando até que o usuário digite 0. Se escolher 1 ou 2, exiba a mensagem correspondente usando `if-else`.

### 14. Contador de Números no Intervalo [10, 20]

Peça ao usuário para digitar 10 números. Ao final do laço `for`, informe quantos desses números estavam dentro do intervalo entre 10 e 20 (inclusive).

### 15. Fatorial Simples

Peça um número ao usuário e calcule o seu fatorial utilizando um laço `while` ou `for`. Exemplo: 5! = 5 x 4 x 3 x 2 x 1 = 120.
